Cat Listing Website System

Developed a responsive and dynamic Cat Listing System using the Ionic Framework (Angular) as an individual project. The system includes a user-friendly interface with a grid layout using ion-grid, ion-row, and ion-col, designed to showcase different cat breeds. Additionally, the application integrates the Cat Facts API to dynamically display breed data, enriching the user experience with interesting facts about each breed.

Weareable Posture Correction System

Led the technical development of an IoT-based wearable posture correction system using a flex sensor and Arduino. The system is designed to detect posture abnormalities in real-time by monitoring the user’s body position and providing feedback to improve posture. My role involved testing, calibrating, and ensuring accurate data collection from the flex sensor, along with developing a real-time feedback mechanism that alerts users when they adopt an incorrect posture.
